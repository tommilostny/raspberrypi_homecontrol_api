# Raspberry Pi Home control API

Local smart home control hub API created using **Python Flask**.

## Hardware
- Raspberry Pi Zero WH
- Breadboard, LEDs, resistors
- Yeelight color bulb
- WOOX WiFi Smart Multi-plug (Tuya Power Strip)
